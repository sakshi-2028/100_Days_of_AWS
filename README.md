## 🚀 About This Repo

Welcome! This is my **#100DaysOfAWS** learning journal. Each day I pick one AWS topic, learn it hands-on, and document everything I understood — *what*, *why*, *steps (console + CLI)*, and notes.

**Goal:** Build a strong foundation in AWS cloud engineering — one day, one concept at a time.
